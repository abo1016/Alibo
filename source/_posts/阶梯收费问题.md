例如：一个用户有11笔订单，则：`30*10 + 1*25 = 55` 以下为计算收费方法

```php
function toll($order, & $toll)
{
    switch($order){
        case $order > 40:
        $toll += ($order - 40)*5 + toll(25, $toll);
        break;
        case $order > 25:
        $toll += ($order - 25)*10 + toll(15, $toll);
        break;
        case $order > 15:
        $toll += ($order - 15)*10 + toll(10, $toll);
        break;
        case $order > 10:
        $toll += ($order -10)*25 + toll(1, $toll);
        break;
        default :
        $toll += $order * 30
        break;
    }
}
toll(60, $toll);
print($toll);
```
